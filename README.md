# Plus Size Thrifted Gems – Website Concept

A premium, responsive e-commerce website concept created for **Plus Size Thrifted Gems**, demonstrating how an Instagram-based fashion business could elevate its online presence with a modern website.

> **Disclaimer:** This is a portfolio concept created by **Nuvanta** for demonstration purposes only. It is not the official website of Plus Size Thrifted Gems.

## ✨ Features

- Responsive, mobile-first design
- Elegant fashion boutique aesthetic
- Hero section with call-to-action
- Featured product showcase
- Product collection page
- About section
- FAQ section
- Contact section
- Floating WhatsApp button
- Smooth scrolling and modern animations

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 🎯 Purpose

This project was created as a portfolio demonstration to showcase how small businesses that currently rely on social media can benefit from having a professional website.
